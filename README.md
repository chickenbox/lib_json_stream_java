# Lib Json Stream Java
> Convert json object to byte array.

Convert json object to byte array. Reduce the size of converted array by reducing redundant property names. 

* Supported data Type:
	- byte,
	- short,
	- int32,
	- float,
	- double,
	- boolean,
	- string,
	- array,
	- object,
	- null

* Limitation:
	- up to 65536 property names.
	- max property name length is 256 bytes.
